## ⚡ Features

  * **Real-Time Tracking:** Track your rank on the ladder without delay.
  * **Performance Metrics:** Keep an eye on your XP/hour.
  * **Gap Analysis:** Instantly see how many experience points you are missing to catch up to the player ahead.
  * **Customizable UI:** The overlay integrates perfectly into your PoE interface.
  * **Low Resource Usage:** Designed to not impact your FPS – while you grind, the tool barely uses any resources.

## 🛠 Installation

1.  Download the latest version from my [Homepage](https://ladder.nichtjfdochichkennedy.com/).
2.  Run the **Exile Ladder Overlay Installer**.

## 🚀 Quick Start

1.  Open **Exile Ladder Overlay**.
2.  Enter your PoE Account name.
3.  Load your characters and select one to track.
